## Getting Started

First, run the development server:

```bash
pnpm install
pnpm dev
```


### Custom className use *ClassName and config extension 

VSCode Tailwind CSS IntelliSense:
 
```json
  "tailwindCSS.classAttributes": [
    "class",
    "className",
    "ngClass",
    ".*ClassName"
  ]
```
